# Madzhab - Aplikasi Fiqih Digital

Aplikasi fiqih digital yang dirancang untuk membantu umat Islam memahami hukum Islam secara ilmiah, netral, dan dewasa, dengan menampilkan berbagai pendekatan fiqih dari empat madzhab utama: Hanafi, Maliki, Syafi'i, dan Hanbali.

- **Empat Madzhab**: Pendapat dari Hanafi, Maliki, Syafi'i, dan Hanbali disajikan secara berimbang
- **Pendekatan Dewasa**: Menghargai perbedaan pendapat sebagai kekayaan keilmuan Islam
- **Mudah Dipahami**: Bahasa sederhana tanpa menghilangkan kedalaman makna dan dalil
- **Desain Menenangkan**: Interface minimalis yang fokus pada kenyamanan membaca
- **Ruang Belajar Khusyuk**: Tanpa kolom debat atau komentar, fokus pada pembelajaran

1. **Thaharah (Bersuci)** - Hukum-hukum bersuci, wudhu, tayammum, dan mandi
2. **Shalat** - Hukum-hukum shalat, waktu, dan tata caranya
3. **Zakat** - Hukum-hukum zakat, nisab, dan distribusinya
4. **Puasa** - Hukum-hukum puasa Ramadan dan puasa sunnah
5. **Muamalah** - Hukum-hukum jual beli, hutang piutang, dan transaksi
